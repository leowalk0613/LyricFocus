## v1.5.3

- **歌词与翻译互换**：仅当当前行有翻译时才互换，修复无翻译时万象息屏歌词不显示
- **长标题歌曲**：优化 `(feat. …)` 等长标题的歌词搜索匹配（如「透明なパレット」）
- **万象息屏长文本**：歌词行绑定前 ellipsize，避免 RemoteViews 超长文本空白
- **万象息屏专辑取色**：按息屏黑底重算对比度，不再沿用锁屏 Monet/文字取色的深色字
- **版本号**：`1.5.3`（versionCode 9）

## 安装说明

- 下载 Release APK 安装（需与上一版相同签名）
- 若曾安装 Debug 版，请先卸载再安装
- 需要 LSPosed，作用域：`com.android.systemui`、`com.miui.aod`
